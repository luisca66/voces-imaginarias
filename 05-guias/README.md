# Guías y Manuales del Estudio

Material de referencia para el equipo de Voces Imaginarias. Aquí viven los manuales, evaluaciones y marcos que usamos durante la migración a un modelo nativo de IA.

Puedes pedirle a tu agente que te explique cualquiera de estos documentos, o abrir la versión `.html` para leerla con buen formato.

## Qué hay aquí

| Documento | Para qué sirve |
| :--- | :--- |
| `manual-cliente` | Tu manual de operación en el día a día con agentes |
| `guia-cliente-mtp` | Cómo definir el Propósito Masivo Transformador del estudio |
| `guia-evaluacion` | Cómo se evalúa la madurez del estudio para la migración |
| `plantilla-scoring-cliente` | La plantilla de diagnóstico (las 7 dimensiones) |
| `marco-gobernanza` | Reglas de supervisión humana sobre los agentes |
| `pasaporte-agente` | Cómo se documentan los límites de cada agente |
| `plantilla-flujos-y-conocimiento` | Cómo mapeamos los flujos de trabajo del estudio |

---

> **Nota:** Estos documentos son copias de referencia. La versión maestra la mantiene Luis (el asesor). Si ves algo desactualizado, avísale para que sincronice la última versión.
